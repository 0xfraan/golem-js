## Prerequisites

- node >= v12.13.0
- yarn >= v1.22.3

## Installation
- run `yarn` in main folder
- add `YAGNA_APPKEY` as environment variable

## Running
- `npm run start` will start blender example by default
- `npm run start:low` will start low level api example
